Design Statement

The design of this website is intentionally minimalistic, ensuring that content remains clear, accessible, and visually engaging. High contrast is used throughout to improve readability and make key elements stand out, enhancing both usability and aesthetic appeal.

A consistent visual identity is maintained through a carefully selected palette of three main colors (pink, blue, and green) and a set of logos that reinforce thematic cohesiveness. This approach ensures that the design remains clean and professional while creating a recognizable and unified experience across pages.

Movement

To create a dynamic and engaging user experience, the website incorporates motion effects powered by the data-aos library. These subtle animations draw attention to important sections, guide navigation, and add a layer of interactivity without overwhelming the content.

Outline

The site is built using an HTML template, specifically HTML5 UP Phantom, allowing for greater flexibility in both formatting and menu navigation. This foundation enables easy customization and adaptation, ensuring that the structure remains scalable and responsive to future content updates.
